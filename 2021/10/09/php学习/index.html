2.echo 指令

输入、输出 指令，echo+空格+’输入内容’ 进行输入

例如 echo 'Hollw Word' ;

例如 echo 'Hollw Word' . "$x";

注释是对于程序的解释，对于 复杂 程序的日后维护、升级、迭代有重要的意义。

单行注释可为 // +内容或 #+内容

多行注释为 /* +内容+ */

4.变量

用于储存数据

与代数类似，可以给 PHP 变量赋予某个值（x=5）或者表达式（z=x+y）。

变量可以是很短的名称（如 x 和 y）或者更具描述性的名称（如 age、carname、totalvolume）。

PHP 变量规则：

  • 变量以 $ 符号开始，后面跟着变量的名称
  • 变量名必须以字母或者下划线字符开始
  • 变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ）
  • 变量名不能包含空格
  • 变量名是区分大小写的（$y 和 $Y 是两个不同的变量）

<1>通过$ +变量名（可以是字母、数字、下划线，但数字不能单独开头作变量）+=定义量

$name='定义量';

特性：(1)若定义如 $b =&$a 时，程序并未开辟新变量，而是直接引用旧变量

(2)可连续使用$$来将已有的变量变为可变变量（类似套娃，作用以后会用到）

<2>检测变量 var_dump(isset($变量名));用来查找变量是否已被定义

<3>删除变量 unset($变量名);

<4>static 声明的变量只会执行一次，值存放在内存当中

<5>PHP 语句和 PHP 变量都是区分大小写的

<6>PHP 是一门弱类型语言

<7>在所有函数外部定义的变量，拥有全局作用域。除了函数外，全局变量可以被脚本中的任何部分访问，要在一个函数中访问一个全局变量，需要使用 global 关键字

<8>在 PHP 函数内部声明的变量是局部变量，仅能在函数内部访问

<9>在不同函数中使用相同的变量名称，因为这些函数内定义的变量名是局部变量，只作用于该函数内

5.函数

是一种语法结构（不会自动运行，需要调用）

函数是通过调用函数来执行的。

你可以在页面的任何位置调用函数。

创建语法

PHP 函数准则：

  • 函数的名称应该提示出它的功能
  • 函数名称以字母或下划线开头（不能以数字开头）

添加参数

为了给函数添加更多的功能，我们可以添加参数，参数类似变量。

函数中存在两种参数，形式参数与实际参数，使得全局变量引入函数内部。

形式参数：在创建函数名称后面的一个括号内指定。形参可以有多个，使用”,”分隔，同时可以给定初始值，当调用函数且无实际参数时将自动执行形式参数的值。

实际参数：在调用函数时在函数名称后面的一个括号内指定。 在调用参数时，系统会将实参传入形参，执行函数

返回值

如需让函数返回一个值，请使用 return 语句。return会结束当前函数。

实例

函数转换进制

八转十 octdec(八进制) 十六转十 hexdec ('十六进制')

更多函数使用手册来查询。

Boolean 类型

是指在PHP中的一种规定好的判断类型，正确与错误，很简单但很实用

字符串

字符串变量用于存储并处理文本

在创建字符串之后，我们就可以对它进行操作了。您可以直接在函数中使用字符串，或者把它存储在变量中

字符串转义：通过在已有某种意义的字符前加上 \使字符正常输出

字符串定界

定义变量后在等号后加 <<<'任意字符'另起一行可以输入任意字符在变量中，结束后另起一行输入刚刚的’任意字符’做结尾即可定界；

字符串函数strlen

用于测定一个变量的长度；如：echo strlen($被测变量)

mb_strlen($被测字符)可测定宽字符；

函数 trim

删除对象变量两端的字符(如空格、“”等)，可以在后面加charlist：'增加的删除值'，只删右边rtrim,只删左边ltrim

trim($目标变量,charlist:'增加的删除值')

函数var_dump

打印变量的相关信息，常用来进行bool类型的判断

函数stroupper

全改大写

函数ucfirst&ucwords

把首字母改大写&把每个首字母都大写

函数md5

将目标编译哈希为十六进制32位的不可逆字符串

函数explode

拆分目标，成为数组（也可查明后缀）

函数implode

合成目标

函数substr

截断字符串如`substr($目标,start:起始在第几个，lenghth:结束在第几个)

7.PHP 类型比较

虽然 PHP 是弱类型语言，但也需要明白变量类型及它们的意义，因为我们经常需要对 PHP 变量进行比较，包含松散和严格比较。

  • 松散比较：使用两个等号 == 比较，只比较值，不比较类型。
  • 严格比较：用三个等号 === 比较，除了比较值，也比较类型

9.条件语句

当编写代码时，常常需要为不同的判断执行不同的动作。可以在代码中使用条件语句来完成此任务。

在 PHP 中，提供了下列条件语句：

  • if 语句

    语句用于仅当指定条件成立时执行代码。

    语法

  • if…else 语句

    在条件成立时执行一块代码，条件不成立时执行另一块代码，使用 if….else 语句。

    语法

  • if…elseif….else 语句

    在若干条件之一成立时执行一个代码块，使用 if….elseif…else 语句。.

    语法

  • switch 语句 - 在若干条件之一成立时执行一个代码块

    有选择地执行若干代码块之一，使用 switch 语句。

    工作原理：首先对一个简单的表达式 n（通常是变量）进行一次计算。将表达式的值与结构中每个 case 的值进行比较。如果存在匹配，则执行与 case 关联的代码。代码执行后，使用 break 来阻止代码跳入下一个 case 中继续执行。default 语句用于不存在匹配（即没有 case 为真）时执行。

    语法

10.数组

数组是一个能在单个变量中存储多个值的特殊变量，并且可以根据键访问其中的值

在 PHP 中，array() 函数用于创建数组：

在 PHP 中，有三种类型的数组：

  • 数值数组 - 带有数字 ID 键的数组

    这里有两种创建数值数组的方法：

    自动分配 ID 键（ID 键总是从 0 开始）如：

    $cars=array("Volvo","BMW","Toyota");

    人工分配 ID 键如：

    $cars[0]="Volvo"; $cars[1]="BMW"; $cars[2]="Toyota";

  • 关联数组 - 带有指定的键的数组，每个键关联一个值

    关联数组是使用您分配给数组的指定的键的数组。

    这里有两种创建关联数组的方法,如：

    $age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");

    or:

    $age['Peter']="35"; $age['Ben']="37"; $age['Joe']="43";

  • 多维数组 - 包含一个或多个数组的数组

数组排序函数

  • sort() - 对数组进行升序排列:元素按照字母升序排列
  • rsort() - 对数组进行降序排列:数组中的元素按照字母降序排列
  • asort() - 根据关联数组的值，对数组进行升序排列
  • ksort() - 根据关联数组的键，对数组进行升序排列
  • arsort() - 根据关联数组的值，对数组进行降序排列
  • krsort() - 根据关联数组的键，对数组进行降序排列

11.循环语句

编写代码时，需要让相同的代码块一次又一次地重复运行，在代码中使用循环语句来完成这个任务。

在 PHP 中，提供了下列循环语句：

  • while - 只要指定的条件成立，则循环执行代码块

    while 循环将重复执行代码块，直到指定的条件不成立。

    语法

  • do…while - 首先执行一次代码块，然后在指定的条件成立时重复这个循环

    语法

  • for - 循环执行代码块指定的次数

    语法

    参数：

    • 初始值：主要是初始化一个变量值，用于设置一个计数器（但可以是任何在循环的开始被执行一次的代码）。
    • 条件：循环执行的限制条件。如果为 TRUE，则循环继续。如果为 FALSE，则循环结束。
    • 增量：主要用于递增计数器（但可以是任何在循环的结束被执行的代码）。

    注释：上面的初始值和增量参数可为空，或者有多个表达式（用逗号分隔）。

  • foreach - 根据数组中每个元素来循环代码块，用于遍历数组

    语法

    每进行一次循环，当前数组元素的值就会被赋值给 $value 变量（数组指针会逐一地移动），在进行下一次循环时，您将看到数组中的下一个值。

    每一次循环，当前数组元素的键与值就都会被赋值给 $key 和 $value 变量（数字指针会逐一地移动），在进行下一次循环时，你将看到数组中的下一个键与值

12.文件

打开文件

定义一个参数，使用fopen()函数将目标文件导入

$定义参数=fopen(”目标文件“,"开启类型”);

模式 描述
r 只读。在文件的开头开始。
r+ 读/写。在文件的开头开始。
w 只写。打开并清空文件的内容；如果文件不存在，则创建新文件。(会清空原有内容，慎用)
w+ 读/写。打开并清空文件的内容；如果文件不存在，则创建新文件。(会清空原有内容，慎用)
a 追加。打开并向文件末尾进行写操作，如果文件不存在，则创建新文件。
a+ 读/追加。通过向文件末尾写内容，来保持文件内容。
x 只写。创建新文件。如果文件已存在，则返回 FALSE 和一个错误。
x+ 读/写。创建新文件。如果文件已存在，则返回 FALSE 和一个错误。

如果打开二进制文件，需要在顶层使用header('Content-type:image/文件后缀'); 来声明文件的类型(不然会产生用记事本打开图片的效果)（原理目前不清楚不了解函数），描述”开启类型”时需要加上b的模式

文件中光标(操作位置)的改变

使用函数fseek($被导入的参数,改变至的位置);

写入文件后或是改变光标的位置后，若想输出全部的文件，一定要输入fseek($被导入的参数,0); 不然只会输出光标之后的量

函数filesize()

“全部”……全部（我也不太清楚具体意义）取得文件大小

关闭文件

使用函数fclose($被导入的参数); (一定要打上，不然每刷新一次刚刚的程序就要跑一次)

判断文件权限

使用函数var_dump(is_权限名able(目标文件)); 进行bool的判断

删除文件

使用函数unlink(目标文件);

13.文件包含

将文件引入一个php文件中（在CTF中的应用暂时没有理解，待学习）

函数include '引入文件的路径'; 函数require '引入文件的路径';

除了处理错误的方式不同之外，两者在其他方面都是相同的；

include在错误时，报错，之后的程序可以继续执行（可以通过在前面加上@的方式隐去报错）；

include_once会检索当前文件是否已被包含，如果已被包含，则不会再次包含

require在错误时，报错，全部的程序被停止；

require_once会检索当前文件是否已被包含，如果已被包含，则不会再次包含
